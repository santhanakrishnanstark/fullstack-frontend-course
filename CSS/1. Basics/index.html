CSS Basics

Basic Syntax for CSS:

    Selectors in CSS

  • Element or Tag selector: eg: h1, p, div
  • Class selector: eg: .my-class
  • Id selector eg: #my-id
  • Attribute selector: eg: [property] or we can use both property and value => syntax: [property="value"] eg: [class="text-input"]
  • Pesudo Selector: eg: :pesudo-name { property: value } hover me to see pesudo css effect

CSS Implementations or way to use css in HTML document

  • Inline Styles => if the css is added inside the tag itself
  • Internal Styles => if the css is added inside style tag in a same html file is called => Internal CSS
  • External Stylesheets => to create new css file and link with HTML document using link tag
