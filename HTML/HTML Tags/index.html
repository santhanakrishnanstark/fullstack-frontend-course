HTML Tags

Wrapper tags

  • Div - Division tag (it is a container for flow contnet)
  • header - to wrap the header content.
  • main - to wrap the main content of the page
  • footer - to wrap the footer content
  • section - we can use both div or section tag to create a group of content.

Attributes

  • id - an Id name should not be repeated, one unique name for per page.
  • class - It used to specify a class for html element, we can have same class name in multiple places. it used to apply same style for different tags.
