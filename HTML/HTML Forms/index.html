  • Radio button - by using radio button we can select only one option from the radio button group. (If the number of options is limited or small only we can use checkbox)
  • Checkbox - by using checkbox we can select multiple values from checkbox group. (If the number of options is limited or small only we can use checkbox)
  • Dropdown/select - if the number of options is exceeded more than 10 then we can use dropdown to select single/multiple values
  • readonly - make the field not editable but it is focuable
  • disabled - make the field not editable and can't be focused.
  • selected - to set default select option for select tag
